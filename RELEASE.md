## 安装与运行

1. 双击 `EasyUCD-Setup.exe`，解压出 `EasyUCD` 文件夹。
2. 进入文件夹双击 `EasyUCD.exe`，浏览器自动打开界面。关掉黑色命令行窗口即停止运行。
3. 另需装一个本机 AI 助手负责生成网页，**二选一**：[Claude Code](https://docs.claude.com/claude-code) 或 [OpenCode](https://opencode.ai)；登录后或在右上角「模型配置」填好模型信息即可。

开发者也可用源码运行：`npm install && npm run build && npm start`，然后访问 http://localhost:5273 。自己打安装包：`npm run package`。

**前置要求**：Windows 10/11（安装包方式）；源码方式需 Node 18+。

## 这个版本能做什么

- **两种画界面方式**：左侧控件库点选摆放，或在「文字生成线框图」里一句话自动生成线框，可混用。
- **一键生成 HTML**：点「生成 HTML」，几秒得到真实网页，支持下载 `.html` 与复制源码。
- **近 91 个精绘控件**，按用途分 10 类（容器、导航、输入、表单进阶、选择进阶、展示、反馈与状态、数据展示、动作、媒体），由单一数据源统一生成。
- **可视化点选编辑器**：在右侧预览里直接点选元素，用属性面板改文字、颜色、字号、边框、间距、尺寸等样式。
- **自定义控件库**：把画好的组合「添加到资源库」，转存为可复用控件。
- **控件库快速搜索 / 过滤**，按分类归档。
- **双 AI 引擎**：Claude Code 与 OpenCode 自由切换；支持自定义模型接入与可配置生成超时。

## 已知问题与提示

- **生成需要本机 AI 助手**：点了「生成 HTML」没反应或失败，多半是 Claude Code / OpenCode 没装好或没登录，检查后在右上角切换重试。
- **生成耗时**：一般十几秒；选更精细的成品风格会久一些，默认最多等 5 分钟。
- **想空跑流程不接 AI**：源码启动时加 `WTH_MOCK=1`，生成返回占位内容、不调用真实模型。
- **平台**：当前安装包仅提供 Windows 版。
